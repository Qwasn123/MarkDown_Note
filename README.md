## 1. 标题 📌
使用 `#` 来表示标题，`#` 的数量代表标题级别。

```markdown
# 一级标题
## 二级标题
### 三级标题
```


## 2. 列表 📝
### 无序列表
使用 `-` 或 `*` 来创建无序列表。

```
- 第一项
- 第二项
- 第三项
```

### 有序列表
使用数字和 `.` 来创建有序列表。

```
1. 第一项
2. 第二项
3. 第三项
```

## 3. 字体样式 ✨
### 加粗
使用 **文本** 或 __文本__ 表示加粗。

```
**加粗文本**
```

### 斜体
使用 *文本* 或 _文本_ 表示斜体。

```
*斜体文本*
```

### 删除线
使用 ~~文本~~ 表示删除线。

```
~~删除线文本~~
```

### 组合样式
可以组合使用多种样式：

```
***加粗且斜体***
~~**加粗且删除**~~
*~~斜体且删除~~*
```

效果：
***加粗且斜体***
~~**加粗且删除**~~
*~~斜体且删除~~*

## 4. 链接 🔗
使用 `[显示文本](链接地址)` 来创建链接。

```
[百度](https://www.baidu.com)
```

### 内部链接
可以链接到文档内的其他部分：

```
[跳转到标题部分](#标题)
```

### 引用式链接
使用引用式链接可以让文档更整洁：

```
[百度][1]
[GitHub][2]

[1]: https://www.baidu.com
[2]: https://github.com
```

### 图片链接
可以将图片和链接结合使用：

```
[![GitHub](https://github.githubassets.com/images/modules/logos_page/GitHub-Mark.png)](https://github.com)
```

## 5. 图片 🖼️
使用 `![图片描述](图片地址)` 来插入图片。

```
![图片示例](https://www.baidu.com/img/bd_logo1.png)
```

### 图片尺寸调整
使用 HTML 标签可以调整图片尺寸：

```
<img src="图片地址" width="200" height="100">
```

## 6. 引用 💬
使用 `>` 来创建引用，可以有多个层级。

```
> 这是一级引用
>> 这是二级引用
>>> 这是三级引用
```

### 引用中的格式化
引用中可以使用其他 Markdown 语法：

```
> **加粗文本**
> - 列表项
> 1. 有序列表
```

## 7. 代码块 ��
使用 ``` 包裹代码块。

```python
print("Hello, world!")
```

### 行内代码
使用单个反引号来标记行内代码：

```
使用 `print()` 函数来输出内容
```

### 代码块语法高亮
可以在开头指定语言来启用语法高亮：

```javascript
const greeting = "Hello World";
console.log(greeting);
```

## 8. 表格 📊
使用 `|` 来分隔单元格，使用 `-` 来分隔表头和内容。

```
| 表头1 | 表头2 | 表头3 |
| ----- | ----- | ----- |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |
```

### 表格中的格式化
表格单元格中也可以使用其他 Markdown 语法：

```
| 名称 | 描述 |
|------|------|
| *斜体* | **加粗** |
| `代码` | [链接](url) |
```

## 9. 任务列表
使用 `- [ ]` 来表示未完成的任务，使用 `- [x]` 来表示已完成的任务。

```
- [ ] 任务1
- [x] 任务2
- [ ] 任务3
```

## 10. 分隔线
使用三个或更多的 `-`、`*` 或 `_` 来创建分隔线：

```
---
***
___
```

## 11. 转义字符
使用反斜杠 `\` 来转义特殊字符：

```
\* 这不是斜体 \*
\# 这不是标题
```

## 12. 注脚
可以使用注脚来添加参考说明：

```
这里是一段文字[^1]
[^1]: 这里是注脚的内容
```

## 13. HTML
Markdown 中可以直接使用 HTML 标签：

```
<div align="center">
  <p>居中的段落</p>
</div>
```
